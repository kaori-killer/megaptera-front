## 데이터

백엔드에서 JSON 형태의 데이터를 돌려주는 API를 제공한다. 대부분 REST API 또는 GraphQL을 쓴다.
프론트는 이 데이터를 사용자가 볼 수 있도록 UI를 구성한다.

리액트는 선언형(HTML과 유사한 모양의 DSL을 사용)으로 UI를 구성할 수 있다.
